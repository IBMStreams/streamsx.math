list<float64> distanceList(list<float64> list1, list<float64> list2)

Compute the (one-dimensional) distance of list elements. The distance is computed with (abs(list2[i]-list1[i])). The size of the result is the minimum of the size of input lists. (min(size(list1), size(list1)))

Parameters

  • list1: input value list 1
  • list2: input value list 2

Returns

  • A list that contains the distance between input values
